yellowMontreal WordPress Theme

Description:
Un thème WordPress responsive avec une section Hero entièrement personnalisable via le Customizer.
It is based on FontAwesome 5 and the NeueMontreal font.

Installation:
1. Uploadez le dossier du thème dans `wp-content/themes/`
2. Activez le thème depuis le tableau de bord WordPress
3. Personnalisez la Hero Section via `Apparence > Personnaliser > Section Hero`

Fonctionnalités:
- Design responsive
- Section Hero modifiable depuis le Customizer
- Code optimisé pour la personnalisation
